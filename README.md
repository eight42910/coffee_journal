# ☕ Coffee Journal

バニラ JavaScript で作るコーヒー記録アプリ。

- 記録（豆・焙煎度・温度・抽出秒数・評価・メモ）
- 検索・ソート・平均スコア
- localStorage 保存

## 起動方法

1. VSCode で `index.html` を右クリック → 「Default Browser で開く」または Live Server 拡張機能を使用
2. 記録を追加すると自動的に localStorage に保存されます。
3. ブラウザでリロードしてもデータは残ります。

## 学習テーマ

- フォーム操作とバリデーション
- 配列メソッド（map/filter/reduce/sort）
- localStorage 永続化
- 一方向データフロー（入力 → 状態更新 → UI 再描画）
